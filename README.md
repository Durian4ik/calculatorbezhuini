короч как установить эту залупу:
1. качаешь архив, распаковываешь.
2. Прежде всего у тебя должен быть постгресикуэл.
3. В сикуэле надо сначала подключится на сервер, потом создать дата базу addition_lab.
4. после разархивации открываешь проект а потом запускаешь app.py (перед этим установи всякие библиотеки недостающие, там flask и еще чет)
5. в созданной дата базе будут созданы таблицы для хранения информации.
6. затем переходишь по адрессу который выведет консоль обычно http://127.0.0.1:5001/calculate
7. ну а дальше пару плотных двоечек по монитору и клавиатуре и калькулятор без хуйни запустится.
8. если пишет 22erno ошибка фейл аргументс и тд тп, в коде меняешь порт. типо было 5001, будет 5002.
9. если проблема не уйдет, удали калькулятор, он тебе не нужен.

Руководство по использованию:
это калькулятор, он нужен чтобы считать слова, цифры.
циферки надо писать только с пробелами.
слова надо писать с пробелами и каждый первый символ это большая буква в слове. иначе я не смог заставить эту шайтан машину отделить слово от слова.
там будет табличка создаваться с правильными результатами, можешь на нее не смотреть.

весь код был написан самостоятелно без использования ии.
